To run the program from command line, go to the root directory and 1. compile /Main.java:
		javac kmeansclustering/Main.java
2. create directories results/index
where index = k_<kvalue>
3. After this, run Main.class:
java kMeansclustering/Main

The output will be shown on the window and confusion matrix will be saved in test/results/index/confMat.txt
